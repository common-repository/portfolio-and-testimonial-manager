The best portfolio and testimonial manager with jquery based filter for portfolios. Also a testimonial scroller to use in any section of your page. 

== Description ==

A simple plugin that handles both portfolios and testimonials with ease. You can manage both portfolio and testimonial in a single post and show either of them on your pages.

We provide you with 2 shortcodes to show up both portfolio and testimonial separately.

[portfolios]
This would list the Portfolio image and name with jquery based filter option for categories. On clicking the portfolio image or name, you will be taken to the portfolio detail page which lists client information, portfolio description along with testimonials.

[testimonials]
This will show up only the testimonials in a scroller, which can be used in any part of your website pages.

== Installation ==

1. In your WordPress admin panel, go to *Plugins > New Plugin*, search for *Portfolio and Testimonial Manager* and click "Install now"
1. Alternatively, download the plugin and upload the contents of `portfolio-and-testimonial-manager.zip` to your plugins directory, which usually is `/wp-content/plugins/`.
1. Activate the plugin

== Frequently Asked Questions ==

= How to display portfolio in posts or pages? =
Use the `[portfolios]` shortcode.

= How to display testimonials in posts or pages? =
Use the `[testimonials]` shortcode.
